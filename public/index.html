Smart Waste Management

Real-Time Tracking, Real-World

"JunkRe" is a smart waste management web application revolutionizing urban waste handling. Real-time garbage truck tracking allows users to monitor truck locations and ensures timely service.

The smart waste management web application not only streamlines waste collection processes but also educates and engages the community, leading to more sustainable urban environments and contributing to the achievement of SDG 11.

JunkRe also gamifies responsible waste disposal, rewarding citizens with points for proper segregation and recycling. These points can be redeemed for incentives, fostering community engagement and promoting sustainable practices.
